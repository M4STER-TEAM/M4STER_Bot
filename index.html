M4STER_Bot

Bot desarrollado por @M4STER_ANGEL para Telegram con una API basada en .lua

El bot no está en la branch master

El bot se encuentra en la branch beta puesto que aún no está acabado al 100%

Instalación

1 Instalar librerías

sudo apt-get update
sudo apt-get install lua5.2
sudo apt-get install lua-sec
sudo apt-get install lua-socket
sudo apt-get install lua-cjson

2 Clonar el repositorio

git clone -b beta https://github.com/M4STERANGEL/M4STER_Bot.git
cd M*

3 Configurar el bot

Tendrás que configurar el archivo config.lua, en donde pone bot_api_key, tu token que te dió el BotFather

4 Iniciar el bot

./launch.sh

5 (Opcional) Actualizar el repositorio

git pull
